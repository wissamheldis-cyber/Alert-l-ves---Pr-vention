## Rouge

Le rouge n’est pas décoratif. Il sert à :

- CTA principal ;
- statut actif ;
- point de vigilance ;
- soulignement court ;
- micro-accent de navigation.

Maximum conseillé : 5 % de l’écran.

## Formes

- cartes : 18 px ;
- grands blocs : 26 à 34 px ;
- boutons : 12 px ;
- icônes : trait simple, noir ou rouge ;
- séparateurs : lignes fines, gris ou rouge court.

## Typographie

Base recommandée :

- titres : Inter / Satoshi / Suisse Int’l ;
- corps : Inter ;
- fallback : system UI.

Hiérarchie :

- H1 : 56–78 px desktop ;
- H2 : 36–56 px ;
- H3 : 20–24 px ;
- corps : 16–20 px ;
- labels : 11–13 px uppercase.
